## Color Palette

Single 10-stop blue→slate gradient:

| Stop | Hex | Usage |
|---|---|---|
| 1 | `#0466c8` | Smart Blue — primary start |
| 2 | `#0353a4` | Steel Azure |
| 3 | `#023e7d` | Regal Navy |
| 4 | `#002855` | Prussian Blue |
| 5 | `#001845` | Deep Prussian |
| 6 | `#001233` | Midnight |
| 7 | `#33415c` | Twilight Indigo |
| 8 | `#5c677d` | Blue Slate |
| 9 | `#7d8597` | Slate Grey |
| 10 | `#979dac` | Cool Steel — accent |

CSS custom properties (`--background`, `--foreground`, `--primary`, `--secondary`, `--accent`, etc.) are derived from the current gradient position via `derivePalette()` — no per-scene hue shifts.

- **Body:** Inter (sans-serif)
